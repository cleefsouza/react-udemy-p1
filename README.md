# :pushpin: Projeto 1 - React Udemy :pushpin:

> Projeto 1 - Curso de React.Js + Next.Js completo do básico ao avançado.

### Requisitos
- Node.js
- YARN | NPM

### Tecnologias Usadas
- React.Js

### Executando
- Clone o projeto:
  ```shell
  git@github.com:cleefsouza/react-udemy-p1.git
  ```

- Instale as dependências:
  ```shell
  yarn -i
  # ou
  npm install
  ```

- Executando api:
  ```shell
  yarn start
  # ou
  npm run start
  ```
